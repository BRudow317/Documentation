### Arithmetic Operators
- `**` **Exponentiation** - Raises left operand to the power of right operand.
- `*` **Multiplication** - Multiplies two operands.
- `/` **Division** - Divides left operand by right operand.
- `%` **Remainder** - Returns the remainder of division.
- `+` **Addition** - Adds two operands.
- `-` **Subtraction** - Subtracts right operand from left operand.

### Increment and Decrement
- `--` **Decrement** - Decreases operand by one.
- `++` **Increment** - Increases operand by one.
- `A++` **Postfix Increment** - Increments operand by one and returns the value before increment.
- `A--` **Postfix Decrement** - Decrements operand by one and returns the value before decrement.
- `++A` **Prefix Increment** - Increments operand by one and returns the value after increment.
- `--A` **Prefix Decrement** - Decrements operand by one and returns the value after decrement.

### Unary Operators
`()` **Grouping Operator** - Groups expressions for precedence control.
`,` **Comma Operator** - Allows multiple expressions in a single statement and returns the last result.
`void` **Void Operator** - Evaluates an expression and discards its return value.
`+` **Unary Plus** - Converts its operand to Number type.
`-` **Unary Negation** - Converts operand to Number type and then negates it.

### Relational Operators
- `!` **Logical NOT** - Logical NOT operator.
- `==` **Equality** - Returns true if operands are equal (with type coercion).
- `!=` **Inequality** - Returns true if operands are not equal (with type coercion).
- `===` **Strict Equality** - Returns true if operands are equal and of the same type.
- `!==` **Strict Inequality** - Returns true if operands are not equal or not of the same type.
- `<` **Less Than** - Returns true if left operand is less than right operand.
- `>` **Greater Than** - Returns true if left operand is greater than right operand.
- `<=` **Less Than or Equal** - Returns true if left operand is less than or equal to right operand.
- `>=` **Greater Than or Equal** - Returns true if left operand is greater than or equal to right operand.

### Logical Operators
- - `? :` **Ternary Conditional** - Returns one of two values based on the logical value of the condition.
- `&&` **Logical AND** - Returns true if both operands are truthy (short-circuits).
- `||` **Logical OR** - Returns true if either operand is truthy (short-circuits).
- `??` **Nullish Coalescing** - Returns right operand if left operand is null or undefined.
- `&` **Bitwise AND** - Performs bitwise AND operation on operands.
- `|` **Bitwise OR** - Performs bitwise OR operation on operands.
- `^` **Bitwise XOR** - Performs bitwise XOR operation on operands.

### Bitwise Operators
- `~` **Bitwise NOT** - Bitwise NOT operator.
### Bitwise Shift Operators
- `<<` **Bitwise Left Shift** - Shifts all bits of operand left by specified positions.
- `>>` **Bitwise Right Shift** - Shifts all bits of operand right by specified positions.
- `>>>` **Unsigned Right Shift** - Shifts all bits of operand right by specified positions, filling with zeros.

### Assignment Operators
- `=` **Assignment** - Assigns value to left operand.
- `*=` **Multiplication Assignment** - Multiplies and assigns.
- `/=` **Division Assignment** - Divides and assigns.
- `%=` **Remainder Assignment** - Calculates remainder and assigns.
- `+=` **Addition Assignment** - Adds and assigns.
- `-=` **Subtraction Assignment** - Subtracts and assigns.
- `<<=` **Left Shift Assignment** - Performs left shift and assigns.
- `>>=` **Right Shift Assignment** - Performs right shift and assigns.
- `>>>=` **Unsigned Right Shift Assignment** - Performs unsigned right shift and assigns.
- `&=` **Bitwise AND Assignment** - Performs bitwise AND and assigns.
- `^=` **Bitwise XOR Assignment** - Performs bitwise XOR and assigns.
- `|=` **Bitwise OR Assignment** - Performs bitwise OR and assigns.
- `**=` **Exponentiation Assignment** - Performs exponentiation and assigns.
- `&&=` **Logical AND Assignment** - Assigns if left operand is truthy.
- `||=` **Logical OR Assignment** - Assigns if left operand is falsy.
- `??=` **Nullish Coalescing Assignment** - Assigns if left operand is null or undefined.

## Classes & Modules & Packages

### Modules

- `import` **Import Statement** - Imports bindings from another module.
- `export` **Export Statement** - Exports bindings to be used in other modules.
- `export default` **Export Default Statement** - Exports a single value or entity as the default export of a module.
- `export * from` **Re-export Statement** - Re-exports all exports from another module.
- `export { name1, name2 as alias2 } from` **Named Re-export Statement** - Re-exports specific exports from another module.
- `import.meta` **Import Meta** - An object exposing context-specific metadata to a JavaScript module.
- `super` **Super Keyword** - Calls the parent constructor or allows accessing parent object properties.
- `import()` **Dynamic Import** - Allows loading a module asynchronously and dynamically.

### [Classes](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Class)
- `this` **This Keyword** - Refers to a special property of an execution context.
- `class` **Class Expression** - Defines a class expression.

#### Constructors
- `new` **New Operator** - Creates an instance of a constructor.

#### Inheritance
- `extends` **Extends Keyword** - Indicates that a class is a child of another class.
